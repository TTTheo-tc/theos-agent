# TheOS 1.0.0

Release date: 2026-03-28

## Summary

This is the initial TheOS repository baseline.
Version history has been reset for the new codebase, and `1.0.0` is the first
release point for this repository.

## Highlights

- Unified TheOS naming across runtime, docs, templates, and metadata
- Versioned internal documentation structure for core runtime, integrations,
  tools, platforms, and learning systems
- Unified developer routing through `BOT.md`
- Initial release baseline for changelog and future versioned releases

## Breaking Changes

- Previous pre-TheOS version lineage is not carried forward in this repository
- `CHANGELOG.md` now starts from the TheOS `1.0.0` baseline

## Migration Notes

- Treat this repository as a fresh starting point
- Use `CHANGELOG.md` for ongoing engineering changes
- Use `RELEASE.md` for version-cut summaries only
