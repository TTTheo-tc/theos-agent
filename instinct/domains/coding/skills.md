# skills Domain

## Keywords
skill, skills, 技能, find skill, 找skill, 查skill, 装skill, install skill, 搜skill, skill包, plugin, 插件, 扩展, extension

## Skills
- find-skills: discover and install agent skills from the skills.sh ecosystem
- clawhub: search and install agent skills from ClawHub registry

## Context
查找或安装 skill 时，优先用 find-skills（skills.sh 生态）或 clawhub（TheOS 生态）。
用 `npx skills find <query>` 搜索，`npx skills add <pkg> -g -y` 安装。
安装后需要重启 session 才能加载新 skill。
