# github Domain

## Keywords
github, git, pr, pull request, merge, branch, issue, issues, ci, workflow, action, commit, repo, repository, clone, push, review, 代码审查, 提交

## Skills
- github: manage PRs, view CI status, create/close issues, list workflow runs

## Context
优先用 `gh` CLI 而不是网页操作。创建 PR 前先确认是否已有同目的分支。
CI 失败时用 `gh run view --log-failed` 直接看失败步骤。
